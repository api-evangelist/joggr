# Joggr (joggr)

Joggr is a documentation platform purpose-built for software teams. It bridges the gap between traditional dev docs and modern development workflows, helping engineering teams create and maintain technical documentation. Joggr ships native MCP integration with AI coding agents (Claude Code, Cursor, Windsurf) and APIs for building custom Slackbots, agents, and workflows on top of the platform. The primary domain has moved from joggr.io to joggr.ai.
